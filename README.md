# LIS545termproject-whalesandearthquakes
Term project for Winter '26 LIS 545A: Data Curation Fundamentals

## Introduction

This repository is the final result of a term project for LIS 545B Data Curation: Principles at the University of Washington. The dataset stored in this repository, found on Figshare, is entitled _Shaken, not stirred: blue whales show no acoustic response to earthquake events_ and was authored by Dawn R. Barlow, Mateo Estrada Jorge, Holger Klinck, and Leigh G. Torres. This dataset also accompanies a research article of the same name that was published in Volume 9, Issue 7 of the Royal Society Open Science journal on July 13, 2022. 

## Files Included in Repository 

- Metadata: XML metadata file adhering to DataCite metadata schema
- Final Report: includes final Data and Metadata Profile, final PANGAEA Repository Profile, and any other Additional Details relevant to the data.
